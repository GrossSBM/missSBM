# missSBM: Handling Missing Data in Stochastic Block Models

When a network is partially observed (here, NAs in the adjacency matrix
rather than 1 or 0 due to missing information between node pairs), it is
possible to account for the underlying process that generates those NAs.
'missSBM', presented in 'Barbillon, Chiquet and Tabouy' (2022)
[doi:10.18637/jss.v101.i12](https://doi.org/10.18637/jss.v101.i12) ,
adjusts the popular stochastic block model from network data sampled
under various missing data conditions, as described in 'Tabouy,
Barbillon and Chiquet' (2019)
[doi:10.1080/01621459.2018.1562934](https://doi.org/10.1080/01621459.2018.1562934)
.

The missSBM package provides the following top-level functions:

## Details

- [`observeNetwork`](https://grosssbm.github.io/missSBM/reference/observeNetwork.md)
  a function to draw a partially observe network from an existing, fully
  observed network according to a variety of sampling designs

- [`estimateMissSBM`](https://grosssbm.github.io/missSBM/reference/estimateMissSBM.md)
  a function to perform inference of SBM from a partially observed under
  various sampling designs.

- [`missSBM_param`](https://grosssbm.github.io/missSBM/reference/missSBM_param.md)
  a function to define the list of parameters controlling
  [`estimateMissSBM`](https://grosssbm.github.io/missSBM/reference/estimateMissSBM.md)

These function leads to the manipulation of a variety of R objects
instantiated from some R6 classes, with their respective fields and
methods. They are all generated by the top-level functions itemized
above, so that the user should generally not use their constructor or
internal methods directly. The user should only have a basic
understanding of the fields of each object to manipulate the output in
R. The main objects are the following:

- [`missSBM_fit`](https://grosssbm.github.io/missSBM/reference/missSBM_fit.md)
  an object that put together an SBM fit and and network sampling fit -
  the main point of the missSBM package !

- [`missSBM_collection`](https://grosssbm.github.io/missSBM/reference/missSBM_collection.md)
  an object to store a collection of missSBM_fit, ordered by number of
  block

- [`SimpleSBM_fit_MNAR`](https://grosssbm.github.io/missSBM/reference/SimpleSBM_fit_MNAR.md)
  an object to define and store an SBM fit with MNAR values

- [`SimpleSBM_fit_noCov`](https://grosssbm.github.io/missSBM/reference/SimpleSBM_fit_noCov.md)
  an object to define and store an SBM fit without covariate, MAR values

- [`SimpleSBM_fit_withCov`](https://grosssbm.github.io/missSBM/reference/SimpleSBM_fit_withCov.md)
  an object to define and store an SBM fit with covariates, MAR values

- [`networkSampling`](https://grosssbm.github.io/missSBM/reference/networkSampling.md)
  an object to define and store a network sampling fit

missSBM extends some functionality of the package sbm, by inheriting
from classes and methods associated to simple stochastic block models.
